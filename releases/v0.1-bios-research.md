# v0.1-bios-research

## DBYTE A780L3B BIOS 95W Research Snapshot

This release note captures the key BIOS artifacts tracked under `bios/` and the main research result from the A780L3B / 78LDPxxx 95W threshold investigation.

This is a research snapshot, not a public firmware support release.

## Main Finding

The useful path is the threshold strategy:

```text
100000 -> 130000
A0 86 01 00 -> D0 FB 01 00
```

The older branch-skip path is deprecated:

```text
428-S95 branch skip at 0x019249
```

## Postmortem

During later testing, the board entered a failed POST state after a flash attempt reportedly powered off unexpectedly during the flashing process.

Observed state:

```text
No video output
POST beep pattern
System powers off after a short time
USB recovery did not start
```

Working diagnosis:

```text
Interrupted or incomplete BIOS write / bad flash state.
```

Recovery direction:

```text
Use stock BIOS only.
Use CH341A / SPI programmer or a repair shop capable of flashing the BIOS chip.
Do not continue testing modified firmware until the board is recovered.
```
